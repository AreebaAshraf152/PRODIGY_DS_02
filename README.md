## â–¶ï¸ How the Code Works
1. **Upload Dataset**: The code asks you to upload a Titanic dataset CSV in Google Colab.  
2. **Data Exploration**: It displays dataset head, tail, summary statistics, and column info.  
3. **Missing Values Handling**:  
   - Drops rows where "Embarked" is missing.  
   - Fills missing "Cabin" values with `"Unknown"`.  
   - Replaces missing "Age" values with the mean age.  
4. **Duplicates Check**: Identifies and reports duplicate rows.  
5. **Visualizations**:  
   - **Age Distribution**: Histogram with KDE.  
   - **Gender Count**: Male vs Female distribution.  
   - **Scatter Plot**: Age vs Fare relationship.  
   - **Survival by Gender**: Survival counts based on gender (if "Survived" column exists).
